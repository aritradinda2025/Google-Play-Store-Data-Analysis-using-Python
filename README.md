# Google-Play-Store-Data-Analysis-using-Python
This project analyzes mobile application data from the Google Play Store using the Google Play Store Apps Dataset to identify trends in installs, ratings, reviews, and categories. The analysis was performed using Python with libraries like Pandas and Plotly to generate interactive visualizations and insights.


Project Overview

This project analyzes mobile application data from the Google Play Store to identify trends in app installs, ratings, reviews, and user sentiment. The analysis is based on the Google Play Store Apps Dataset and focuses on understanding factors that influence app popularity and user engagement.

🎯 Objectives

Analyze trends in app installs and ratings
Explore relationships between app size, installs, and ratings
Study user sentiment from reviews
Create interactive visualizations to understand app performance


🛠️ Tools & Technologies

Python
Pandas
NumPy
Plotly
Jupyter Notebook

📊 Visualizations

The project includes several interactive charts:
Bubble Chart (App Size vs Rating vs Installs)
Choropleth Map (Global installs by category)
Stacked Area Chart (Cumulative installs over time)
Time Series Chart (Install trends across categories)

🔎 Key Insights

Apps with higher ratings often have higher installs.
Certain categories like entertainment and communication attract more users.
User sentiment plays an important role in app popularity.
